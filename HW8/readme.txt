Features Fully Implemented:
    -Single line scrabble boards with two double word score bonus tiles and tow double double letter scores tiles.
    -Used tile are removed from avaible tils.
    -Tracking of remaining tiles, current word, and score. 
    -Score tracking with rules adding a bnous 50 points if the user used all 7 tiles.
    -Tiles are permantly placed once places, and tiles can only be drop next to previously placed tile except for the first tile.
    -User cannot replace placed tiles with new tiles in hand.
    -New tiles are added to user's rack to a maximum of 7 total tiles after submiting a word.
    -A new game can be start by clicking the reset game button.
    -Tiles deal to the user are generated randomly from the current available tiles.

Features Not Yet Implemented:
    -Full Scrabble board implementation with all lines and vertical placement.
    -Verification of word using dictionary.
    -Multiplayer rules are not implemented such as remaining tiles scoring at the end of the game.
